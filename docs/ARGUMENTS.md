# Launch arguments for TerraM4KC

| Args      | Description                                        | Variable name |
| --------- | -------------------------------------------------- | ------------- |
| --debug   | Launches the game with ImGui Debug mode            | g_debug_mode  |
| --opengl  | Launches the game with OpenGL                      | g_use_opengl  |
| --mcpi-on | Launches the game with a listener for the MCPI Api | g_mcpi_on     |

## !!OPENGL AND MCPI API SUPPORT ARE CURRENTLY EXPERIMENTAL!!
